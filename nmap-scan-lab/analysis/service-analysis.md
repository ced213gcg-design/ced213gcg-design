# Service Analysis

## Observed Open Ports
- Port 22 (SSH)
- Port 80 (HTTP)
- Port 443 (HTTPS)

## Analysis
Each open port represents an active service and a potential entry point.

### SSH (22)
- Remote access service
- Risk if weak credentials or exposed externally

### HTTP (80)
- Web service
- Risk if unpatched or misconfigured

### HTTPS (443)
- Encrypted web traffic
- Still vulnerable if certificates or backend are weak

## Analyst Perspective
Exposure does not equal compromise, but increases attack surface.
