# Packet Analysis Lab

## Objective
Analyze network traffic to identify suspicious behavior using packet-level inspection.

## Scenario
A system generated unusual outbound traffic. This lab simulates how an analyst investigates packets to determine if the activity is benign or malicious.

## Workflow
1. Capture or review packet data
2. Identify source and destination
3. Analyze protocol behavior
4. Detect anomalies or patterns
5. Document findings

## Tools Used
- Wireshark
- Network protocol analysis

## Files
- analysis/packet-breakdown.md
- analysis/findings.md

## Skills Demonstrated
- Packet inspection
- Network protocol understanding
- Threat identification
- Analytical documentation
